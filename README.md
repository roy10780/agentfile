# AGENTFILE

El Dockerfile para agentes IA

Describe tu idea en una sola frase y AGENTFILE construye el agente completo automáticamente.

## Ejemplo

Abre el terminal y escribe:

idea: Quiero un agente que invierta inteligentemente en la bolsa de Nueva York, automatice trading, analice riesgos y me envíe reportes diarios por WhatsApp.

## Qué hace automáticamente

- Rowboat → Lee tu idea y crea el grafo de agentes con memoria
- Oh My ClaudeCode + Claw Code → Equipo de agentes que escriben código juntos
- MiroFish → Simula miles de escenarios
- LangGraph → Planifica todo
- OpenClaw → Ejecuta en el mundo real (WhatsApp, trading, etc.)
- PicoClaw → Corre en hardware barato
- LiteLLM + MCP → Cualquier LLM y herramientas
